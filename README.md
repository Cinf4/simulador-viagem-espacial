# 🚀 Simulador de Missão Espacial

Este é um projeto desenvolvido em **Python** para calcular o tempo de deslocamento entre astros. 

### 🛠️ O que aprendi com este código:
- **Entrada e Saída de dados:** Interação com o usuário via terminal.
- **Tratamento de Erros:** Uso de `try/except` para evitar que o programa trave com dados inválidos.
- **Lógica Matemática:** Conversão de unidades (Km/h para Dias).

### 📖 Como rodar:
1. Tenha o Python 3 instalado.
2. Execute o arquivo: `missao_simulator.py`
